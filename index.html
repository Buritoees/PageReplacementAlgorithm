Algorithm Information

Understanding page replacement algorithms

FIFO

First-In, First-Out

Replaces the oldest page in memory, regardless of usage frequency.

Complexity O(1)
Structure Queue
  • Simple implementation
  • Low overhead
  • Belady's Anomaly

LRU

Least Recently Used

Replaces the page that hasn't been accessed for the longest time period.

Complexity O(1)
Structure Stack/Map
  • Optimal performance
  • No anomaly
  • More complex
Understanding Page Faults A page fault occurs when a referenced page is not in memory and must be loaded from disk. Efficient algorithms minimize page faults to improve system performance.
